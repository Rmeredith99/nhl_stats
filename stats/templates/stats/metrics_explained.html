What is a metric?

Metrics are a way of measuring players based on values that aren't in the table. With metrics you can create custom stats which gives an alternate way to rank and analyze players.


How do you write a Metric?

Metrics are written using a mostly math-based language which you can write directly into the metric box. The language is quick and intuitive while remaining quite capable.

To use a statistic category in your metric, simply use the name of the stat as if it were a variable name. Capitalization and spacing are not important when writing a metric; "gamesPlayed", "Games Played", and "Ga MeSpL aYeD" are all equivalent. The language is capable of math operations such as addition, subtraction, multiplication, division, and exponentiation as well as aggregate functions like average, sum, min, max, and standard deviation which act on the entire set of players in the filtered group. In order to keep metrics clean and readable, you can assign values to variables for later user.

There is a standard structure for a metric; the first lines contain variable assignments while the final line contains the value to be returned. There is no explicit return command. Instead, the value from the final line is always the one that is returned.


Examples
  • goals
  • Returns the player's goals
  • playerHeight + 6
  • Returns the player's height plus 6
  • missedShotsOverNet - 10
  • Returns the player's total missed shots over the net minus 10
  • assists * 2
  • Returns twice the number of assists
  • goalsPer60Minutes / 60
  • Returns goalsPer60Minutes divided by 60
  • plusMinus ^ 2
  • Returns the square of the player's plus-minus
  • avg(points)
  • Returns the average points over all players in the filter set
  • sum(points)
  • Returns the total points summed over all players in the filter set
  • min(points)
  • Returns the smallest value of points for all players in the filter set
  • max(points)
  • Returns the largest value of points for all players in the filter set
  • std(points)
  • Returns the standard deviation of points for all players in the filter set
  • n = goals - 10
    (assists + 5.4) * n
  • Assigns the player's goals minus 10 to the variable n then returns that value times the quantity assists plus 5.4
    Note that the final line is the only value that is returned
